Automação com PyAutoGUI — Aula 1 (Hashtag)

✅ Status: Projeto concluído

📋 Descrição

Script de automação desenvolvido em Python utilizando a biblioteca pyautogui, referente à Aula 1 do curso da Hashtag Programação. O script automatiza ações repetitivas de mouse e teclado (como navegação entre campos com Tab, cliques e digitação), simulando interações que normalmente seriam feitas manualmente.

🚀 Tecnologias utilizadas


Python 3.14
PyAutoGUI — automação de mouse e teclado
